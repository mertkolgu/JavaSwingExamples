# Dil Seçimi

Java Swing kullanarak RadioButton, ButtonGroup ve ComboBox hakkında bir örnek uygulamadır. Apache Netbeans 11.1 IDE kullanılarak yazılmıştır. Jar dosyası oluşturulurken maven-jar-plugin kullanılmıştır.
