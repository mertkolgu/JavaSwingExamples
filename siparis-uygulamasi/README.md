# Sipariş Uygulaması

Java Swing kullanarak yemek siparişi seçimi yapan örnek uygulamadır. Apache Netbeans 11.1 IDE kullanılarak yazılmıştır. Jar dosyası oluşturulurken maven-jar-plugin kullanılmıştır.
