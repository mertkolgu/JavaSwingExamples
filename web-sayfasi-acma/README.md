# Web Sayfası Açma

Java Swing kullanarak istediğimiz web sayfasını açan örnek uygulamadır. Apache Netbeans 11.1 IDE kullanılarak yazılmıştır. Jar dosyası oluşturulurken maven-jar-plugin kullanılmıştır.
